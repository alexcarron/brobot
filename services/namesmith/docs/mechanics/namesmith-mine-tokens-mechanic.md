# Namesmith Mine Tokens Mechanic
## How Often
Anytime
## How Much
Awards a random amount based on heavy tail distribution
- Minimum of 1 token
- Expected value of 1.5 tokens overtime
- No maximum number of tokens (Could award 1000s but it is extremely unlikely because of the distribution)
## How
Running `/mine` in the `#mine-tokens` channel
## Timing
Currently, if spammed, you can do 20 /mines in aroun ~58 seconds
