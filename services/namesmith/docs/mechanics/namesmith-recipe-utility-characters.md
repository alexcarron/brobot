# Recipe Utility Characters
Utility characters function like normal characters except that you'll find them used in recipes a LOT! They represent actions takes on the characters like rotating, flipping, or shifting them to create a new character. They're main purpose is to be used in character crafting recipes to get the character you want. Character recipes where the input characters don't simply match the output character can use utility characters to get the outcome desired. In terms of game logic and code, they function exactly the same. The only difference is their frequent use in recipes.
## Rotate ↻
Rotates the character(s) 90°, 180°, or 270°
- N → Z
- b → q
## Flip ⇋
Mirrors the character(s) horizontally or vertically
- < → >
- \[ → \]
## Lean ⤻
Subtly rotates the character(s) at an acute angle
- / → |
## Shift ↹
Move a character in the set horizontally or vertically
- cI → q
## Fuse ⚭
Combines the characters into the same space as a unified character
- /= → ≠
- o+ → ⊕
## Split ✂
Seperates the character into characters
- ▢ → \[\]
- ≠ → /=
- % → o/o
## Scale ⤡
Changes the size of the character(s)
- o → ○
- TM → ™
## Stretch ⇔
Elongate the character(s) vertically or horizontally
- - → —
- o → 0
